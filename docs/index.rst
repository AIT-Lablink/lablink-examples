************************************
Getting started with the AIT Lablink
************************************

.. meta::
   :description lang=en: AIT Lablink examples

This documentation provides step-by-step instructions for simple yet instructive example setups (clients, configuration, scripts) for running the `AIT Lablink <https://ait-lablink.readthedocs.io>`_.
These examples are intended to showcase the basics of Lablink usage to new users, enabling them to start developing their own Lablink applications and setups.
